# üìù Todo App (Vanilla JavaScript)

A clean and minimal **Todo application** built with **Vanilla JavaScript**, focusing on core frontend concepts without frameworks.

This project was created as a learning exercise to understand how state, DOM manipulation, and persistence work under the hood.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using **localStorage**
- Light / Dark mode toggle
- Clean, modern UI with CSS variables
- Scrollable task list for larger collections

---

## Tech Stack

- **HTML5**
- **CSS3**
  - CSS Variables
  - Flexbox
- **Vanilla JavaScript (ES6+)**
- **localStorage**
